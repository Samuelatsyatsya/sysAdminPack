# sysAdminPack
 DevOps Environment Setup

## Overview
This project contains a simple Bash script to set up a basic DevOps project environment.  
It creates directories for logs, configuration files, and scripts, adds sample files, and sets appropriate permissions.

## Setup Instructions
1. Clone the repository:
git clone <repo-url>
cd <repo-name>

##Make the script executable:
chmod +x setup_environment.sh

###Run the script:
./setup_environment.sh
